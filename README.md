# Project DarkElixir
> Redmi Note 5 Pro (Whyred) development.

Project DarkElixir is a modern discovery platform built for the Redmi Note 5 Pro (Whyred) community. It brings together Custom ROMs, Kernels, Recoveries, Firmware, Magisk Modules, and installation guides into one organized experience.

---

## Download Policy

Every download button redirects to the original developer or official hosting platform.

DarkElixir does **not** host, modify, mirror, or redistribute ROMs, kernels, recoveries, firmware, or any other downloadable files unless explicitly stated.

All rights belong to their respective developers and maintainers.
